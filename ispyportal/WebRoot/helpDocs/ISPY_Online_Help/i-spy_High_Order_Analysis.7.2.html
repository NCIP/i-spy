High Order Analysis Overview
The High Order Analysis function enables you to perform the following analyses:
•
Performing a Class Comparison
•
Performing a Principal Component Analysis
•
Performing Hierarchical Clustering Analysis
•
Performing Correlation Scatter Plot Analysis
•
Performing Categorical Plot Analysis
•
Specifying GenePattern Input
Report results are listed on the View Results page (see High Order Analysis Results).
