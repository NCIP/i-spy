Creating a PatientDID List from a Clinical Report
On any Clinical page, you can select and save patients to a PatientDID list.
1.
There are two ways to select patients on the Clinical window:
–
To select an individual, select the box in the I-SPY ID column.
–
To select all of the patients, select the All box .
To clear all of the patients, uncheck the All box.
2.
To save the patients to a PatientDID list, enter a name for the list. You can select PatientDIDs to further filter a query or perform an analysis.
3.
Click the save checked button. Sample List Saved appears.
4.
Click the OK button.
The new PatientDID list is now displayed in red in the blue panel at the bottom of the PatientDID names. Mouse over the name and the data items appear.
