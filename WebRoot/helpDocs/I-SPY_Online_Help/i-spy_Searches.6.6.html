Performing a p53 Query
A p53 query enables you to generate p53 mutation reports using customized search criteria. The search criteria filter the report based on patient groups, timepoints, p53 mutation status, and mutation types.
To define a p53 query, follow these steps:
1.
On the Search tab, click the p53 Query button. The p53 Query Form opens.
2.
Select the appropriate search criteria.
You must enter a name for the search (at the bottom of the page). The name must be unique among all the queries in the current session. Selecting other criteria is optional.
The following table lists the available search criteria. To select more than one option in a list box, Shift-click or Ctrl-click.
