ID Lookup Overview
The ID Lookup function enables you to find information about samples for a given patient by entering either sample or patient identifiers (see Looking Up a Patient or Sample Identifier). Once you perform the lookup, you can also perform the following tasks:
•
Displaying/Hiding the Patient Sample Information
•
Downloading Patient Sample Information to an Excel File
•
Creating a PatientDID List with the ID Lookup
Note:
You can use a PatientDID list to filter I-SPY queries and perform data analysis.
