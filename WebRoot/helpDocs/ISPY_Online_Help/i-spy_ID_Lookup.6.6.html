Creating a PatientDID List with the ID Lookup
From the ID Lookup page, you can save multiple patientsâ€™ data to a PatientDID list. You can use PatientDID lists to further filter a query or analyze data. To create an I-SPY PatientDID list, follow these steps.
1.
Select the box next to each patient to be saved to the PatientDID list or select the All box to select all of the patients.
2.
Name the list.
3.
Click the save button.
The list name appears in red in the side bar at the bottom of the PatientDID Lists.
To further modify the new PatientDID list, see Managing Lists.
