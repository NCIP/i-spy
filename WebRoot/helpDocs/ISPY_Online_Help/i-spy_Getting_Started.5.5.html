Welcome to I-SPY 1.0
The I-SPY workspace comprises a set of five tabs, a blue panel, help links, and a logout link. The five tabs enable you to perform the following functions:
1.
Perform an ID lookup.
2.
Perform complex searches.
3.
Perform higher order analyses.
4.
View results of the searches and analyses.
5.
Manage lists.
The blue panel displays pre-defined patient identifier lists and the default gene identifier list. As you add your own lists, the new lists will appear in red.
