Getting Help
Information about how to use I-SPY is easily accessed from I-SPY’s menu in the top left of the I-SPY workspace.
The following table describes each item on the I-SPY menu.
Help
How to Access
Complete online help
To access the complete version of online I-SPY help, click the help linklocated in the I-SPY menu.
For complete page-level help, click on any I-SPY page.
For brief field help, click .
Application support
To obtain support for I-SPY, click the support link located under the I-SPY menu.
User’s Guide
To access a pdf version of the I-SPY User’s Guide, click the user guide link located under the I-SPY menu.
Download Integrate Data File
To download the Integrated Data File, click the integrated data file link. The file is an Excel spreadsheet comprising all of the clinical data for I-SPY integrated with the MRI data. The file also containes all of the patient to sample ID mappings for each data type.
For application support, see Application Support.
