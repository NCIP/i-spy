High Order Analysis Overview
The High Order Analysis function enables you to perform the following analyses:
•
Class Comparison Analysis (see Performing a Class Comparison)
•
Principal Component Analysis (PCA) (see Performing a Principal Component Analysis)
•
Heiarchical Clustering Analysis (see Performing Hierarchical Clustering Analysis)
•
Correlation Scatter Plot Analysis (see Performing Correlation Scatter Plot Analysis)
•
Categorical Plot Analysis (see Performing Categorical Plot Analysis)
Report results are listed on the View Results page (see High Order Analysis Results).
