Managing Lists Overview
The I-SPY Manage Lists function centralizes all activities pertaining to the creation and management of user-defined, as well as study-defined PatientDID Lists and Gene Lists. With these lists, you can further refine queries or facilitate analysis. For more information, see PatientDID Lists and Gene Lists.
Using the Manage List function, you can perform the following functions:
•
View data items in a list (see Viewing the Data Items in a List)
•
Create new lists from existing lists:
•
Removing Data Items to Create a New List
•
Combining Existing Lists to Create a New List
•
Delete lists (see Deleting an Entire List)
•
Add lists by uploading them or typing them (see Adding a New “Custom” List)
