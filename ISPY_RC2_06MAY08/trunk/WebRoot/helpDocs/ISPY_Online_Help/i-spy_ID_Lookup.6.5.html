Downloading Patient Sample Information to an Excel File
From the ID Lookup page, you can download one patient’s sample data to a file or download all the listed patients’ data to the same file. Follow these steps.
To download an individual patient’s data to an Excel file, follow these steps:
1.
Click next to the patient for which you want to download data.
2.
Name the file and select a location.
The individual patient’s data is saved to the Excel file.
If you searched for multiple patients, to save all the patients’ data to the same Excel file, follow these steps.
1.
Click above the list of patients to download all the patient sample information to one file.
2.
Name the file and select a location.
All the patients’ data is saved to the same Excel file listed in descending order by patient identifier.
